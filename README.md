# Contoh Kasus Pemrograman Go

Folder ini berisi kumpulan contoh kode dan proyek kecil dalam bahasa pemrograman Go. Setiap subfolder merepresentasikan implementasi atau konsep tertentu.

## Daftar Isi

* `clean_architectur`: Contoh implementasi Clean Architecture.
* `CLI_todo_app`: Aplikasi Todo sederhana berbasis Command Line Interface (CLI).
* `custom_err`: Contoh pembuatan tipe error kustom.
* `database_management`: Contoh manajemen koneksi database.
* `fizz_buzz`: Solusi untuk masalah klasik FizzBuzz.
* `frekuesi_kata`: Program untuk menghitung frekuensi kata dalam teks.
* `hitungLuas`: Program sederhana untuk menghitung luas.
* `kalkulator_variadic`: Kalkulator yang menggunakan fungsi variadic.
* `mini-sistem-pembayaran`: Sistem pembayaran mini.
* `paymentGetWay`: Contoh implementasi gateway pembayaran.
* `paymentGetway2`: Contoh lain dari implementasi gateway pembayaran.
* `pointer_swap`: Contoh penggunaan pointer untuk menukar nilai.
* `RestAPI_middleware`: Contoh penggunaan middleware pada REST API.
* `struct`: Contoh penggunaan struct.
* `struct2`: Contoh lain dari penggunaan struct.
* `Table_drive_test`: Contoh implementasi table-driven test.
* `variable_tipe_data`: Contoh penggunaan variabel dan tipe data.
